HTML refers to Hypertext Mark-up Language developed by Tim Berners Lee in 1993. HTML is used to make static webpages that can be read through different browsers.

Structure of HTML

The basic structure of HTML includes:

Doctype:< DOCTYPE html>
HTML:<html>
Head: <head>
Title: <title>Title of the Page goes here </title>
Closing Head tag: </head>
Body: <body>
Contents goes here...........
Closing Body: </body>
Closing HTML: </html>
