Comments, Horizantal Rule and Break in HTML

HTML comments are visible to anyone that views the page source code,but are not rendered when the HTML document is rendered by a browser. An HTML comment begins with <!-- and the comment closes with -->.
Horizontal rules are used to provide a visual break and divide content. We can give horizontal rule in an HTML using <hr> tag.
Break is used in HTML to insert a single line break.
We use <br> tag to force line break in HTML.
